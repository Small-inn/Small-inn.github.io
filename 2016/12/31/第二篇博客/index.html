Angular JS 介绍

1.定义：是一款前端的MVC框架
2.与jQuery的区别：jQuery是一个类库（一系列函数的集合）以DOM为驱动（核心），
  而Angular是一个框架（一系列的类库的集合），以数据和逻辑为驱动

3.AngularJS核心：模块化、双数据绑定、语义化标签、依赖注入

4.与之相似的还有React、vue等框架

MVC

MVC是一种开发模式（Model、View、Controller）
模型：一般用来处理数据（读取/设置）,一般操作数据库
视图：一般用来展示数据,听过HTML展示
控制器：一般是用作链接模型和视图的桥梁

模块化

AngularJS构建应用时是以模块化的方式组织的，
将整个应用化成若干模块，各个模块都有自己的各自职责，然后组成一个整体


定义应用:ng-app属性，表示此标签所包裹的内容都属于App的一部分

定义模块：AngularJS提供一个全局对象angular，此全局对象存在若干方法， 、
其中，angular.module()方法定义一个模块（其实App本质是一个模块大的模块）

定义控制器：作为链接模型和视图的桥梁存在，所以当定义好了控制器以后也就定义好了模型和视图

模型数据是要展示导视图上的，所以需要将控制器关联到视图上，
通过HTML标签加ng-controller属性并且赋值响应的控制器的名称，确定关联关系

具体操作：
第一步：引入angular包
第二步：创建模板
第三步：创建自己的代码

var App = angular.module('App',[])

App.controller('Demo',['$scope',function($scope){
    //自己的代码
    $scope.name = '客店小二LX'

}])

body ng-app = 'App'

div ng-constroller='Demo'

h1 {{name}}

指令

1.ng-app：指定一个应用
可以应用任意标签上，并将此标签区视为一个应用的范围，
一个页面可以有多个但是不能嵌套，一般制定一个

2.ng-controller：指定某视图隶属某一个控制器
var App = angular.mudule(‘App’,[]);

App.controller(‘DemoController’,[$scope,function($scope){

}])

3.ng-show:当属性值为true时显示，false时不显示
4.ng-hide:当属性值为true时隐藏，false时显示
5.ng-if:当属性值为true时显示，false时不显示
5.ng-src:增强地址
6.ng-href增强地址
7.ng-class控制类名
8.ng-include引入模板
9.ng-disabled表单禁用
10.ng-readonly表单只读
11.ng-checked单/复选框表单选中
12.ng-selected下拉框表单选中
13.自定义指令
通过angular全局对象下的directive方法实现
//代码：
var App = angular.module(‘Demo’, []);

// 自定义控制器
App.controller('ItcastController', ['$scope', function ($scope) {

    $scope.list = ['html', 'css', 'js'];

}]);

// 自定义指令
// 第一个参数，指令的名称
// 第二个参数，回调函数
App.directive('itcast', function () {
    // 此回调函数指定具体指令应该做哪些事情
    // 通过一个对象来定义

    var obj = {
        // 自定义指令可以是属性可以是标签
        // E element
        // A attribute
        // C class
        // M mark
        restrict: 'ECMA',
        // 所有内容必须包含在一个根元素下
        template: '<div><h1>你好, AngularJS!</h1><p>哈哈</p></div>',
        // templateUrl: './header.html',
        replace: true,
        // scope: {},
        // link: function () {},
        // compile: function () {}
    }

    return obj;
});
